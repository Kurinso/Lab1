# Lab1
1. Цель работы
Разработка единого класса на языке C#, содержащего методы для решения 20 различных задач, с организацией дружественного интерфейса в методе Main и обеспечением проверки корректности вводимых пользователем данных.

2. Решаемые задачи
В работе были реализованы следующие 20 задач:

Сумма последних цифр - вычисление суммы двух последних цифр числа

Проверка положительного числа - определение, является ли число положительным

Проверка заглавной буквы - определение, является ли символ заглавной буквой

Проверка делимости - определение, делит ли одно число другое нацело

Многократное сложение - последовательное сложение последних цифр пяти пар чисел

Безопасное деление - деление с защитой от деления на ноль

Строка сравнения - формирование строки с результатом сравнения двух чисел

Тройная сумма - проверка возможности получить третье число из суммы двух других

Склонение возраста - правильное склонение слова "год" в зависимости от числа

Вывод дней недели - вывод указанного дня и всех последующих дней недели

Числа наоборот - вывод чисел от заданного до нуля

Возведение в степень - вычисление степени числа

Одинаковость цифр - проверка, все ли цифры в числе одинаковы

Левый треугольник - вывод треугольника из звездочек

Угадайка - игра на угадывание случайного числа

Поиск последнего вхождения - поиск последнего вхождения элемента в массиве

Добавление в массив - вставка элемента в массив по указанной позиции

Реверс массива - обратный порядок элементов массива

Объединение массивов - конкатенация двух массивов

Удаление отрицательных элементов - создание нового массива без отрицательных элементов

3. Структура программы
3.1. Основной класс AllTasksSolver
Класс содержит все 20 публичных методов, соответствующих сигнатурам из задания, и несколько приватных вспомогательных методов.

3.2. Публичные методы
Каждый метод решает конкретную задачу:

Имена методов соответствуют сигнатурам из задания

Все методы имеют модификатор доступа public

Методы возвращают типы данных, указанные в сигнатурах

3.3. Приватные вспомогательные методы
Для организации дружественного интерфейса реализованы:

GetIntegerInput(string message) - безопасный ввод целых чисел

GetCharInput(string message) - безопасный ввод символов

GetStringInput(string message) - ввод строк

GetArrayInput(string message) - ввод массивов с указанием размера

PrintArray(int[] arr, string message) - форматированный вывод массивов

3.4. Метод Main
Организован как центральный управляющий метод:

Последовательный вызов всех 20 методов

Понятные приглашения к вводу для каждого метода

Форматированный вывод результатов

Обработка всех сценариев выполнения

4. Особенности реализации
4.1. Проверка входных данных
Для всех методов, требующих ввода данных с клавиатуры, реализована валидация:

Целые числа: проверка через int.TryParse()

Символы: проверка длины вводимой строки

Массивы: проверка корректности размера

Позиции в массивах: проверка границ диапазона

Дни недели: проверка соответствия списку допустимых значений

4.2. Обработка граничных случаев
Деление на ноль: возврат 0 вместо исключения

Отрицательные числа: использование Math.Abs() где необходимо

Пустые массивы: корректная обработка нулевого размера

Некорректные позиции: возврат исходного массива без изменений
